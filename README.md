# SyncTools

一个基于Go语言开发的同步工具，提供图形用户界面。

## 功能特点

- 图形用户界面
- 文件同步功能
- 错误日志记录
- 崩溃恢复机制

## 技术栈

- Go语言
- lxn/walk (GUI框架)

## 安装和使用

1. 克隆仓库
```bash
git clone https://github.com/YOUR_USERNAME/synctools.git
```

2. 进入项目目录
```bash
cd synctools
```

3. 运行程序
```bash
go run cmd/server/main.go
```
